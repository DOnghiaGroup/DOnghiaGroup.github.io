Code

Most of our code is accessible via our GitHub page. Public repos are listed below.

Magellanic Corona Snapshots
Simulation snapshots used in the Lucchini et al. (2020) paper showing the formation and evolution of the Magellanic Stream including the effects of a Magellanic Corona. Snapshots are in HDF5 format from a GIZMO simulation and are formatted the same as GADGET snapshots.
If you make use of any of these snapshots, please cite Lucchini, S. et al. Nature 585, 203-206. (2020).
Magellanic Astropy Plugin
A plugin for Astropy that allows for conversion into the Magellanic Coordinate system using the transform_to function.
If you make use of this plugin please list GitHub repo in code section.
SNN Clustering Algorithm
A machine learning algorithm to detect clustering of groups in multidimentional data.
If you make use of this plugin please list GitHub repo in code section.
MGwave
A wavelet transformation Python module that can be used to identify structures of varying sizes in images.
If you make use of this software please cite Wavelet paper.
